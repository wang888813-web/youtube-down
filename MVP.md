## 一、产品概述

### 1.1 产品定位

面向 YouTube 内容创作者和研究者的 AI 工具套件。用户粘贴 YouTube 链接，系统自动提取字幕并通过 AI 生成摘要、脚本或关键要点，同时支持视频/音频下载。

### 1.2 目标用户

| 用户类型 | 核心诉求 |
|----------|----------|
| 内容创作者 | 快速获取视频脚本，二次创作素材 |
| 学生 / 研究者 | 提炼视频核心内容，节省观看时间 |
| 营销人员 | 分析竞品视频内容，提取关键信息 |

### 1.3 核心价值主张

> 粘贴一个 YouTube 链接，30 秒内获得 AI 摘要、完整字幕和可下载脚本。

## 二、MVP 范围

### 2.1 核心功能（Must Have）

#### F1 — AI 脚本生成器（主功能）
- 输入：YouTube 视频 URL（含 Shorts）
- 处理：调用字幕提取 API → AI 摘要生成
- 输出：
  - 视频摘要（200字以内）
  - 关键要点列表（3-5条）
  - 完整字幕文本（带时间戳）
- 支持导出：复制到剪贴板 / 下载 .txt

#### F2 — YouTube Shorts 视频下载
- 输入：Shorts URL
- 输出：无水印 MP4 文件下载
- 支持画质选择：720p / 1080p

#### F3 — MP3 音频提取
- 输入：YouTube 视频 URL
- 输出：MP3 文件下载

### 2.2 次要功能（Should Have，二期）

- 多语言字幕支持
- 用户账号系统（登录 / 注册）
- 使用次数限制 + 付费解锁（Freemium）
- YouTube 趋势分析工具

### 2.3 不做（Won't Have in MVP）

- 批量处理
- 浏览器插件
- 视频编辑功能
- 社交分享

## 三、用户流程

### 主流程（AI 脚本生成）

```
用户访问首页
    ↓
粘贴 YouTube 链接
    ↓
点击「Generate」
    ↓
系统提取字幕（调用 YouTube Transcript API）
    ↓
AI 处理生成摘要 + 要点
    ↓
展示结果页（摘要 / 字幕 / 下载）
```

### 下载流程

```
用户粘贴链接 → 选择格式（MP4 / MP3）→ 点击下载 → 文件输出
```

## 五、技术方案

### 5.1 技术栈建议

| 层级 | 方案 |
|------|------|
| 前端 | Next.js + Tailwind CSS |
| 后端 | Node.js / Python FastAPI |
| 字幕提取 | youtube-transcript-api（开源）或 RapidAPI |
| AI 摘要 | OpenAI GPT-4o / Claude API / 火山引擎 |
| 视频下载 | yt-dlp（开源） |
| 部署 | Vercel（前端）+ Railway/Render（后端） |

### 5.2 关键 API 依赖

- **字幕提取：** `youtube-transcript-api`（Python 开源库）
- **AI 处理：** 大模型 API（按 token 计费）
- **视频下载：** `yt-dlp` 命令行工具

### 5.3 免费额度限制实现

- 未登录用户：IP 限流，每日 5 次
- 登录用户（免费）：每月 50 次
- 付费用户：无限制

## 九、风险与注意事项

| 风险 | 说明 | 应对 |
|------|------|------|
| YouTube 反爬 | yt-dlp 可能被封禁 | 使用代理池 / 备用 API |
| 版权问题 | 下载功能存在法律灰色地带 | 明确 ToS，仅供个人使用 |
| AI 成本 | 高并发下 token 费用高 | 设置免费额度上限 |
| 字幕不可用 | 部分视频无字幕 | 提示用户，提供 Whisper 语音转文字备选 |
